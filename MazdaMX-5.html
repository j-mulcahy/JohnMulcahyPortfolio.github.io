Mazda MX-5

Mid-weight Mazda MX-5, Build for Graphical tests in UE5, Tasked to see how far we could push graphics using base UE5 V1.0. Modelled in Maya, Using a combination of tileable materials and custom UV Sets featuring custom High Polys for normal baking. All aspects have been done by me. Has some shadow issues which I'm told is believed to be a bug with UE5.
